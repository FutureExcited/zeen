# Zoro Model Asset

This folder is the expected drop location for the Sketchfab model used by `example-zoro`.

Model: ZORO  
Author: Boridian  
Source: https://sketchfab.com/3d-models/zoro-554e124c17ee40bfbd4222a1a89e0036  
License: Creative Commons Attribution 4.0, https://creativecommons.org/licenses/by/4.0/

Run `./tools/download-sketchfab-zoro.sh` with `SKETCHFAB_TOKEN` set, or download the model from Sketchfab manually and place `scene.gltf`, `scene.glb`, or `ZORO(finished).obj` here.
